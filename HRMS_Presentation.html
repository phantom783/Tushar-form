üîÑ Data Flow

Employee Registration Flow

  1. User fills out the employee form
  2. Frontend validates data (age, email format, etc.)
  3. Form data sent to backend via POST request
  4. Backend validates and generates employee code
  5. Data saved to MongoDB
  6. Success confirmation shown to user

Employee List Flow

  1. User navigates to employee list page
  2. Frontend fetches employees from API
  3. Data displayed in paginated table
  4. User can edit, view, or delete employees
  5. Changes reflected immediately in UI

üöÄ Getting Started

  • Navigate to backend folder: cd backend
  • Install dependencies: npm install
  • Ensure MongoDB is running on port 27017
  • Start server: npm start
  • Server runs on: http://localhost:5000

Frontend Setup

  • Navigate to react folder: cd react
  • Install dependencies: npm install
  • Start dev server: npm run dev
  • Open browser: http://localhost:5173
